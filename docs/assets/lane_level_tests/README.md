# Lane-Level Diagnostics

This directory contains generated lane-level acceptance evidence. The maintained
runbook is [Lane-Level Acceptance](../../benchmarks/lane-level.md), and the
commit policy is [Benchmark Artifact Policy](../../benchmarks/artifacts.md).

The directory is intentionally committed so lane-level behavior changes are
visible in diffs. Refresh it only from
`scripts/run_lane_level_acceptance_metrics.py`.
