# M4-Style Forecasting

The M4 wrapper runs the maintained M4 sample through the shared forecasting
benchmark harness:

```sh
python scripts/forecasting_m4.py --committed --no-hyperopt
```

This command is a committed sample check, not a full M4 corpus claim. Use
`scripts/forecasting_library_benchmark.py --source m4 --m4-suite
--m4-series-limit 0` before making full-corpus statements.

M4-style scoring should report scale-aware metrics such as sMAPE, MASE, or OWA
where available. RMSE is useful as a diagnostic but is not enough for a public
competition-style quality claim across heterogeneous M4 groups.
