# M5-Style WRMSSE

CartoBoost includes an official-style WRMSSE helper for hierarchical forecasting
validation. The helper is intended for M5-like evaluations where each series has
training history, a forecast horizon, and a value weight.

For each series:

1. Compute the RMSSE scale from squared seasonal naive differences over the
   training history.
2. Compute forecast-horizon mean squared error.
3. Divide horizon MSE by the scale and take the square root.
4. Normalize value weights across all evaluated series.
5. Sum normalized weight multiplied by series RMSSE.

The Python helper is available as `cartoboost.metrics.wrmsse`. It returns a
scalar by default and can return a per-series breakdown with normalized weights,
scales, RMSSE values, and contributions.

This page documents the metric implementation only. It does not claim a new M5
benchmark result. Any future benchmark refresh should record the exact command,
data source, sample size, split definitions, model roster, comparable settings,
metric breakdown tables, timing breakdown, artifact paths, plots, and limitations
alongside the generated evidence.
