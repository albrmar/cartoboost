# Examples Agent Guide

## Dev environment tips
- This folder contains runnable examples and example JSON configs.
- Keep examples small, deterministic, and consistent with the public Python and CLI APIs.

## Testing instructions
- Prefer examples that run from a fresh development environment with `uv run`.
- Exercise changed examples when public API usage changes.

## PR instructions
- Summarize which user workflow the example demonstrates.
- Mention any docs updated to reference changed examples.
